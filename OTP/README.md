# React OTP Input Component

A modern, mobile-friendly OTP (One-Time Password) input UI built using **React** and **Tailwind CSS**.  
Supports typing, backspace navigation, paste detection, and SMS autofill for seamless user experience.

---

## Features

✅ Auto-focus and auto-jump between OTP boxes  
✅ Backspace navigation (clears current or previous input)  
✅ Smart paste handling (splits digits across boxes)  
✅ Mobile-ready (numeric keypad + SMS OTP autofill)  
✅ Fully accessible (keyboard + screen reader support)  
✅ Clean, responsive UI built with Tailwind CSS  

---

## Demo

> A 4-digit OTP input that behaves like native mobile verification screens.  
> Type digits, paste codes, or let your mobile browser autofill automatically.

## Tech Stack

- **React** — Component logic and state management  
- **Tailwind CSS** — Responsive, modern styling  
- **React Hooks** — `useState`, `useRef`, `useEffect`
