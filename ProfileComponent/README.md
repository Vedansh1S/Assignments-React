# ProfileComponent

A responsive dashboard component built with React and Tailwind CSS. Features a collapsible sidebar navigation, user profile card, and webinar schedule display.

## Features

- Responsive sidebar navigation with toggle functionality
- User profile card with contact information
- Webinar schedule with live/upcoming status indicators
- Action buttons for webinar management tasks
- Mobile-friendly design with adaptive layouts

## Getting Started

```bash
npm install
npm run dev
```

The app runs on Vite's dev server (defaults to `http://localhost:5173`).
